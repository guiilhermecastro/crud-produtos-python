# CRUD de Produtos Python

Sistema de cadastro de produtos via terminal desenvolvido em Python com banco de dados SQLite.

## Funcionalidades

- Listar produtos
- Adicionar produto
- Buscar produto por nome
- Atualizar produto
- Remover produto

## Tecnologias utilizadas

- Python
- SQLite (sqlite3)

## Como executar

1. Clone o repositório:
   git clone https://github.com/guiilhermecastro/crud-produtos-python

2. Acesse a pasta:
   cd crud-produtos-python

3. Execute o programa:
   python main.py
